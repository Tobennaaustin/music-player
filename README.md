# Music Player

A simple and customizable music player for web applications.

## Features

- Play, pause, and control music playback.
- Next and previous track navigation.
- Progress bar for tracking song playtime.
- Restart the entire playlist.
- In-built songs.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Web browser with HTML5 audio support (modern browsers are recommended).
- Basic understanding of HTML, CSS, and JavaScript.
